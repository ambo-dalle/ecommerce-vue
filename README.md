# ecommerce-vue

#### Run Server
```sh
$ cd ecommerce-vue/server
$ npm install
$ npm start
```


#### Items
| Route        | Http           | Descriptin  |
| ------------- |:-------------:| -----:|
| /items      | GET   | Get all the books |
| /items/:id      | GET   | Get  one books |
| /items      | POST   | Create / Add a Book |
| /items/:id     | DELETE      |  Delete a Book|
| /items | PUT      |    Update a Book |
